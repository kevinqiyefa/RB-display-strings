## Instructions:

> 1. `Clone the Repo`
> 
> 2. `npm install`
> 
> 3. `cd server` (cd into the server folder)
> 
> 4. `psql < db_schema.sql` to run the PostgreSQL database schema 
> 
> 5. `cd ..` back to the main folder
> 
> 6. `npm start`
> 
> 7. `npm test` to run the unit test and snapshots test
